    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get Refugio up and running in minutes with our step-by-step guide.
  </p>

  <Steps>
    <Step title="Set up your database">
      Create a PostgreSQL or MySQL database and set up the schema. See the [Database Setup](/deployment/database-setup) guide for detailed instructions.
      ```bash
      # Create database
      psql -U postgres -c "CREATE DATABASE refugio;"
      ```
    </Step>
    <Step title="Configure your connection">
      Edit the database connection settings in `conexion.php`:
      ```php
      $host = 'localhost';
      $dbname = 'refugio';
      $user = 'postgres';
      $password = 'your_password';
      ```
    </Step>
    <Step title="Start the server">
      Launch the PHP development server:
      ```bash
      php -S localhost:8000
      ```
    </Step>
    <Step title="Log in">
      Access the application at `http://localhost:8000` and log in with the default admin credentials:
      - Email: `admin@hostel.com`
      - Password: `admin123`
      
      <Warning>Change these credentials immediately after first login.</Warning>
    </Step>
  </Steps>
</div>
